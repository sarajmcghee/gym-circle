# Gym Circle

A mobile-friendly gym workout tracker for Sara, Jeff, and Dawna.

## What It Does

- Shows a 45-60 minute workout for each person.
- Uses a dark theme with profile-specific accent colors.
- Lets each user log weights, reps, timed movements, difficulty, and notes.
- Includes a beginner exercise library with form cues and video-search links.
- Saves progress in each browser with `localStorage`.

## Profiles

- Sara: strength + cardio, purple accents
- Jeff: core + cardio, green accents
- Dawna: strength + cardio, blue accents

## GitHub Pages

This is a static site. To publish it from GitHub:

1. Push this folder to a GitHub repository.
2. In the repo, open `Settings > Pages`.
3. Set the source to `Deploy from a branch`.
4. Choose the `main` branch and `/root` folder.
5. Save and use the Pages URL GitHub provides.

No build command is required.
